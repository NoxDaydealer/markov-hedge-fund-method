## Variant: Jarvis Reference Neural Cloud

### Design stance
A dense, cinematic Nox command field that follows the Video-2 direction: glowing neural lattice plus colorful particle-cloud mode, with the visualization as the main experience rather than decoration.

### Key choices
- Layout: left filters/telemetry, dominant center neural/cloud field, right selected-node operations panel.
- Visual language: near-black Jarvis background, cyan/magenta/violet/amber glows, subtle scanner sweep, dense edges and 128 live entities.
- Interaction: switch between `NEURAL LATTICE` and `PARTICLE CLOUD`, filter by domain, click nodes/clusters to update telemetry.
- Data mapping: nodes represent Hermes/Nox agents, Kanban, cron/watchdogs, Obsidian/Fact Store, Discord/Telegram routes, strategy modules, and paper-risk gates.

### Trade-offs
- Strong at: wow-factor, ambient system awareness, matching the dense neural/cloud design reference.
- Weak at: not a production dashboard yet; actions are read-only and the data is embedded mock/realistic rather than live API-bound.
